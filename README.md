# Betting Bot - Telegram бот для анализа ставок на спорт

Telegram-бот, который помогает находить выгодные ставки на спорт, сравнивая коэффициенты разных букмекеров.

## Возможности

- ⚽ **Сравнение коэффициентов** - анализ коэффициентов разных букмекеров
- 💰 **Value-ставки** - поиск завышенных коэффициентов
- 🔔 **Уведомления** - автоматическая отправка о выгодных предложениях
- 🔮 **Прогнозы** - агрегация прогнозов экспертов

## Установка

### 1. Клонируйте репозиторий

```bash
git clone <repository-url>
cd betting-bot
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

### 5. Запустите бота

```bash
python main.py
```

## Получение токена бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в `.env`

## Команды бота

- `/start` - Начать работу с ботом
- Главное меню доступно через инлайн-кнопки

## Как это работает

### Алгоритм поиска Value-ставок

1. **Сбор данных** - бот парсит коэффициенты с нескольких источников
2. **Расчёт справедливого коэффициента** - среднее значение без маржи букмекера
3. **Сравнение** - если коэффициент выше справедливого, это value-ставка
4. **Уведомление** - бот отправляет сообщение о выгодном предложении

### Формула расчёта

```
Value = (справедливый_коэф / предложенный_коэф) - 1

Если Value > 5% - хорошая ставка
Если Value > 10% - отличная ставка  
Если Value > 15% - редкая возможность
```
